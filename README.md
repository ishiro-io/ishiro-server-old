# Ishiro Server

![Version](https://img.shields.io/badge/version-0.1.0-blue.svg?cacheSeconds=2592000)

>🚀 Back-end server for Ishiro made with NestJS

## ✨  Install

```bash
$ yarn
```

## ⚡️ Usage

```markdown
# development
yarn start

# watch mode
yarn start:dev

# production mode
yarn start:prod
```

## ✅  Test

```markdown
# unit tests
yarn test

# e2e tests
yarn test:e2e

# test coverage
yarn test:cov
```
